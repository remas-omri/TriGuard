ERD – Entity Relationship Diagram

This folder contains the final database design for the Biometric Identification System.

The ERD illustrates:
 • All system entities (Users, Devices, Attempts, Alerts, etc.)
 • Primary and Foreign Keys
 • Relationships between tables (1:N, 1:1)
 • Multi-factor verification structure
 • Retry logic support inside the Attempts entity

The design ensures data integrity, normalization, and full traceability of authentication processes.
