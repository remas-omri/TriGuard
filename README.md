# نظام TriGuard: نظام الأمان الحيوي المتكامل

## وصف المشروع
نظام أمان حيوي يعتمد على إشارة **كهربائية القلب (ECG)** كمعرف أساسي، مدعوماً بالوجه والصوت لضمان أقصى درجات التحقق.

## التقنيات المستخدمة (Technical Stack)
* **المتحكمات:** ESP32 (كمتحكم رئيسي لربط الحساسات).
* **اللغات:** C++ (للجهاز)، JavaScript (للسيرفر)، Dart (للتطبيق).
* **الواجهة الخلفية (Backend):** Node.js مع إطار عمل Express.
* **قاعدة البيانات:** SQLite للسجلات السحابية، ونظام LittleFS للتخزين المحلي.
* **تطوير الجوال:** Flutter لبناء تطبيق إدارة الصلاحيات.
* **الواجهة الأمامية للموقع:** HTML5, CSS3, JavaScript.

## الحساسات والمكونات المادية
1. **HuskyLens:** للتعرف على الوجه.
2. **DF2301Q:** لمعالجة الأوامر الصوتية.
3. **Fingerprint Sensor:** (كبديل حالي لـ ECG).
4. **GC9A01:** شاشة تفاعلية لعرض الحالة.

## كيفية تشغيل الكود
* **برمجة المتحكم:** قم بفتح ملف الكود عبر Arduino IDE، وارفع الكود على شريحة ESP32.
* **تشغيل السيرفر:** انتقل إلى مجلد الـ Backend، نفذ `npm install` ثم `node index.js`.
* **تشغيل التطبيق:** افتح مشروع Flutter واستخدم الأمر `flutter run`.
* **المزامنة:** تأكد من إعداد بيانات الواي فاي في الكود ليرتبط الجهاز بالسيرفر تلقائياً.
