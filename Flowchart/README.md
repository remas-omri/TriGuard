System Flowchart

This folder contains the final authentication flow of the biometric identification system.

The flow describes:
 • Device authentication check
 • Multi-factor verification (Face, Voice, ECG)
 • Logging results into verification_results
 • Updating attempts status
 • Retry logic handling
 • Alert generation for failed or unauthorized attempts
 • Audit logging for granted or denied access

The flow ensures secure processing, controlled retries, and full traceability of every access attempt.
